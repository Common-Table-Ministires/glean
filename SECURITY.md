# Security Policy

Glean is early-stage, pre-release software owned by Common Table Ministries. If you find a security issue, please report it privately rather than opening a public issue.

## Reporting a vulnerability

Use GitHub's private vulnerability reporting for this repository (Security tab → Report a vulnerability), or email hello@commontableministries.com.

Please include:
- What you found and where (file/line if you have it)
- Steps to reproduce
- What you'd expect to happen instead

We'll acknowledge reports within a few days. This is a small, mostly-volunteer project, so response time may vary, but every report gets read.

## Scope

Right now this repo contains a macOS desktop prototype used for development, not a shipped product. The same policy applies once the real iOS app ships.
