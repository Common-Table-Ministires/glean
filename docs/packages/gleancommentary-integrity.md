# Integrity — GleanCommentary

This package is the **only** home for bundled open / public-domain commentary.

| Do | Don't |
|----|--------|
| Depend on `GleanCommentary` from the app | Copy `notes.json` into `ios/Glean` |
| Add voices via pack JSON + license row | Scrape commercial study Bibles |
| Sign every excerpt with author + work | Present notes as anonymous “Glean doctrine” |
| Keep excerpts short (1–4 sentences) | Dump whole commentary chapters into feed |

Scripture text stays in **ScriptureCore**. Formation selection stays in **GleanSelection** (`~/Desktop/biblealgo`). Commentary never rewrites either.
